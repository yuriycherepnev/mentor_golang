# Библиотека API

## Автор

### Получение списка авторов

**GET** `/author`

**Ответ**

```json
{
  "success": true,
  "data": [
    {
      "id": 1,
      "name": "Толстой"
    }
  ]
}
```

---

### Получение автора

**GET** `/author/{id}`

---

### Создание автора

**POST** `/author`

```json
{
  "name": "Пушкин"
}
```

---

### Обновление автора

**PUT** `/author/{id}`

```json
{
  "name": "Новое имя"
}
```

---

### Удаление автора

**DELETE** `/author/{id}`

---

## Книга

### Получение списка книг

**GET** `/book`

---

### Получение книги

**GET** `/book/{id}`

---

### Создание книги

**POST** `/book`

```json
{
  "unique_id": "book-123",
  "title": "Название",
  "id_author": 1
}
```

---

### Обновление книги

**PUT** `/book/{id}`

---

### Удаление книги

**DELETE** `/book/{id}`

---

## Операции

### Взять книгу

**POST** `/book/borrow`

```json
{
  "book_id": 1,
  "reader_id": 1
}
```

---

### Вернуть книгу

**POST** `/book/return`

```json
{
  "book_id": 1
}
```

---

## Формат ответа

```json
{
  "success": true,
  "data": {},
  "error": ""
}
```
